package com.ksea.demo1;

 * Created by mexican on 2017/2/7.
 */
public class SwitchDemo {
    public static void main(String[] args) {
        String a = "你好";
        switch (a) {
            case "你好":
                System.out.println("a");
                break;
            default:
                System.out.println("默认值");
                break;

        }

    }
}

案列 JDK1.5 中switch中不支持字符串，并且编译不通过，
Error:(9, 16) java: -source 1.5 中不支持 switch 中存在字符串
  (请使用 -source 7 或更高版本以允许 switch 中存在字符串)
